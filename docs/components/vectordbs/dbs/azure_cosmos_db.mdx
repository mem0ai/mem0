[Cosmos DB](https://learn.microsoft.com/en-us/azure/cosmos-db/) is a No SQL document database that supports vector search capabilities, allowing for efficient high-dimensional similarity searches over large datasets with robust scalability and performance.

### Installation

Azure Cosmos DB support requires additional dependencies. Install them with:

```bash
pip install azure-cosmos
```

## Usage

```python
import os
from mem0 import Memory

os.environ["OPENAI_API_KEY"] = "sk-xx"

config = {
    "vector_store": {
        "provider": "azure_cosmos_db",
        "config": {
            "cosmos_client": "mem0_db",
            "vector_embedding_policy": {
                "vectorEmbeddings":[
                    {
                        "path": "/embedding",
                        "dataType": "float32",
                        "dimensions": 1536,
                        "distanceFunction": "cosine",
                    }
                ],
            },
            "indexing_policy": {
                "indexingMode": "consistent",
                "includedPaths": [{"path": "/*"}],
                "excludedPaths": [{"path": '/"_etag"/?'}],
                "vectorIndexes": [{"path": "/embedding", "type": "flat"}],
                "fullTextIndexes": [{"path": "/text"}],
            },
            "cosmos_database_properties": {},
            "cosmos_collection_properties": {"partition_key": "partition_key_value"},
            "vector_search_fields": {
                "text_field": "description",
                "embedding_field": "embedding",
            },
            "database_name": "vectorSearchDB",
            "collection_name": "vectorSearchContainer",
        }
    }
}

m = Memory.from_config(config)
messages = [
    {"role": "user", "content": "I'm planning to watch a movie tonight. Any recommendations?"},
    {"role": "assistant", "content": "How about thriller movies? They can be quite engaging."},
    {"role": "user", "content": "Iâ€™m not a big fan of thriller movies but I love sci-fi movies."},
    {"role": "assistant", "content": "Got it! I'll avoid thriller recommendations and suggest sci-fi movies in the future."}
]
m.add(messages, user_id="alice", metadata={"category": "movies"})
```

## Config

Here are the parameters available for configuring Cosmos DB as the vector database in Mem0:

| Parameter | Description | Default Value |
| --- | --- | --- |
| cosmos_client | Client used to connect to azure cosmosdb no sql account | |
| vector_embedding_policy | Vector Embedding Policy for the collection | |
| indexing_policy | Indexing Policy for the collection | |
| cosmos_database_properties | Database Properties for the collection | |
| cosmos_collection_properties | Container Properties for the collection | |
| vector_search_fields | Vector Search and Text Search Fields for the collection | `None` |
| database_name | Name of the database to be created | `"vectorSearchDB"` |
| collection_name | Name of the collection to be created | `"vectorSearchContainer"` |
| search_type | CosmosDB Search Type to be performed | `"vector"` |
| metadata_key | Metadata key to use for data schema | `"metadata"` |
| create_collection | Set to true if the collection does not exist | `True` |
| table_alias | Alias for the table to use in the WHERE clause | `"c"` |
| full_text_policy | Full Text Policy for the collection | `None` |
| full_text_search_enabled | Set to true if the full text search is enabled | `False` |
