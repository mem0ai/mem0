CometAPI is a unified API gateway that provides access to 500+ AI models from various providers through a single OpenAI-compatible interface. To use CometAPI models, you need to set the `COMETAPI_KEY` environment variable. You can obtain the CometAPI API key from the [CometAPI Console](https://api.cometapi.com/console/token).

Learn more about CometAPI at [cometapi.com](https://www.cometapi.com/?utm_source=mem0&utm_campaign=integration&utm_medium=integration&utm_content=integration).

## Usage

```python
import os
from mem0 import Memory

os.environ["OPENAI_API_KEY"] = "your-api-key" # used for embedding model
os.environ["COMETAPI_KEY"] = "your-cometapi-key"

config = {
    "llm": {
        "provider": "cometapi",
        "config": {
            "model": "gpt-4o-mini",
            "temperature": 0.2,
            "max_tokens": 2000,
        }
    }
}

m = Memory.from_config(config)
messages = [
    {"role": "user", "content": "I'm planning to watch a movie tonight. Any recommendations?"},
    {"role": "assistant", "content": "How about thriller movies? They can be quite engaging."},
    {"role": "user", "content": "I'm not a big fan of thriller movies but I love sci-fi movies."},
    {"role": "assistant", "content": "Got it! I'll avoid thriller recommendations and suggest sci-fi movies in the future."}
]
m.add(messages, user_id="alice", metadata={"category": "movies"})
```
