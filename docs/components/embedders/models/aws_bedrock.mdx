title: AWS Bedrock
description: "Configure AWS Bedrock as an embedding provider in Mem0 with IAM credentials for Python or TypeScript."
---

To use AWS Bedrock embedding models, you need the appropriate AWS credentials, permissions, and model access. The SDK uses the standard AWS credential provider chain when explicit credentials are not supplied.

### Setup

- Ensure you have model access from the [AWS Bedrock Console](https://us-east-1.console.aws.amazon.com/bedrock/home?region=us-east-1#/modelaccess)
- Configure credentials using a method described in the [AWS documentation](https://docs.aws.amazon.com/sdkref/latest/guide/standardized-credentials.html)
- Set up environment variables for authentication:

```bash
export AWS_REGION=us-east-1
export AWS_ACCESS_KEY_ID=your-access-key
export AWS_SECRET_ACCESS_KEY=your-secret-key
```

For TypeScript, install the optional Bedrock Runtime client:

```bash
pnpm add @aws-sdk/client-bedrock-runtime
```

### Python usage

```python
import os
from mem0 import Memory

os.environ["OPENAI_API_KEY"] = "your-openai-api-key"
os.environ["AWS_REGION"] = "us-west-2"

config = {
    "embedder": {
        "provider": "aws_bedrock",
        "config": {
            "model": "amazon.titan-embed-text-v2:0"
        }
    }
}

m = Memory.from_config(config)
m.add("I prefer science fiction movies.", user_id="alice")
```

### TypeScript usage

```typescript
import { AWSBedrockEmbedder } from "mem0ai/oss";

const embedder = new AWSBedrockEmbedder({
  model: "amazon.titan-embed-text-v2:0",
  awsRegion: "us-west-2",
  embeddingDims: 512,
});

const embedding = await embedder.embed("Mem0 stores long-term agent memory.");
```

### Config

#### Python

| Parameter | Description                            | Default Value                |
| --------- | -------------------------------------- | ---------------------------- |
| `model`   | The name of the embedding model to use | `amazon.titan-embed-text-v1` |

#### TypeScript

| Parameter            | Description                                               | Default Value                |
| -------------------- | --------------------------------------------------------- | ---------------------------- |
| `model`              | Titan or Cohere embedding model ID                        | `amazon.titan-embed-text-v1` |
| `awsRegion`          | AWS region for Bedrock Runtime                            | `AWS_REGION` or `us-west-2`  |
| `embeddingDims`      | Output dimensions for Titan Text Embeddings V2            | Model default                |
| `awsAccessKeyId`     | Explicit access key; omit to use the AWS credential chain | AWS credential chain         |
| `awsSecretAccessKey` | Explicit secret key; required with `awsAccessKeyId`       | AWS credential chain         |
| `awsSessionToken`    | Optional session token for temporary credentials          | None                         |
