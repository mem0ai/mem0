title: Mistral AI

To use Mistral AI embedding models, please obtain the Mistral AI api key from their [console](https://console.mistral.ai/). Set the `MISTRAL_API_KEY` environment variable to use the embedding model as given below in the example.

### Usage

<CodeGroup>
```python Python
import os
from mem0 import Memory

os.environ["MISTRAL_API_KEY"] = "your_api_key"

config = {
    "embedder": {
        "provider": "mistral",
        "config": {
            "model": "mistral-embed"
        }
    }
}

m = Memory.from_config(config)
messages = [
    {"role": "user", "content": "I'm planning to watch a movie tonight. Any recommendations?"},
    {"role": "assistant", "content": "How about a thriller movies? They can be quite engaging."},
    {"role": "user", "content": "Iâ€™m not a big fan of thriller movies but I love sci-fi movies."},
    {"role": "assistant", "content": "Got it! I'll avoid thriller recommendations and suggest sci-fi movies in the future."}
]
m.add(messages, user_id="john")
```

```typescript TypeScript
import { Memory } from 'mem0ai/oss';

const config = {
  embedder: {
    provider: 'mistral',
    config: {
      apiKey: process.env.MISTRAL_API_KEY || '',
      model: 'mistral-embed',
    },
  },
};

const memory = new Memory(config);
await memory.add("I'm visiting Paris", { userId: "john" });
```

</CodeGroup>

### Config

Here are the parameters available for configuring Mistral AI embedder:

<Tabs>
  <Tab title="Python">

    The name of the embedding model to use | `mistral-embed` | |
    `embedding_dims` | Dimensions of the embedding model | `1024` | | `api_key`
    | The Mistral AI API key | `None` |
