VoyageAI provides high-quality reranking models optimized for relevance scoring.

## Models

VoyageAI offers two reranking models:

- **`rerank-2`**: High-quality reranker with best performance (recommended)
- **`rerank-2-lite`**: Faster and more cost-effective option

## Installation

```bash
pip install voyageai
```

## Configuration

```python Python
from mem0 import Memory

config = {
    "vector_store": {
        "provider": "chroma",
        "config": {
            "collection_name": "my_memories",
            "path": "./chroma_db"
        }
    },
    "llm": {
        "provider": "openai",
        "config": {
            "model": "gpt-4o-mini"
        }
    },
    "reranker": {
        "provider": "voyageai",
        "config": {
            "model": "rerank-2",
            "api_key": "your-voyageai-api-key",  # or set VOYAGE_API_KEY
            "top_k": 5
        }
    }
}

memory = Memory.from_config(config)
```

## Environment Variables

Set your API key as an environment variable:

```bash
export VOYAGE_API_KEY="your-api-key"
```

## Usage Example

```python Python
import os
from mem0 import Memory

# Set API key
os.environ["VOYAGE_API_KEY"] = "your-api-key"

# Initialize memory with VoyageAI reranker
config = {
    "vector_store": {"provider": "chroma"},
    "llm": {"provider": "openai", "config": {"model": "gpt-4o-mini"}},
    "reranker": {
        "provider": "voyageai",
        "config": {
            "model": "rerank-2",
            "top_k": 3
        }
    }
}

memory = Memory.from_config(config)

# Add memories
messages = [
    {"role": "user", "content": "I work as a data scientist at Google"},
    {"role": "user", "content": "I specialize in machine learning and NLP"},
    {"role": "user", "content": "I enjoy playing tennis on weekends"}
]

memory.add(messages, user_id="alice")

# Search with reranking
results = memory.search("What is the user's profession?", user_id="alice")

for result in results['results']:
    print(f"Memory: {result['memory']}")
    print(f"Vector Score: {result['score']:.3f}")
    print(f"Rerank Score: {result['rerank_score']:.3f}")
    print()
```

## Configuration Parameters

| Parameter | Description | Type | Default |
| --- | --- | --- | --- |
| `model` | VoyageAI rerank model to use | `str` | `rerank-2` |
| `api_key` | VoyageAI API key | `str` | `None` |
| `top_k` | Maximum documents to return | `int` | `None` |
| `truncation` | Whether to truncate long documents | `bool` | `True` |

## Features

- **High Quality**: State-of-the-art relevance scoring
- **Fast**: Optimized for low latency
- **Truncation**: Automatic handling of long documents
- **Flexible**: Works with any vector store

## Best Practices

1. **Model Selection**: Use `rerank-2` for best quality, `rerank-2-lite` for faster/cheaper processing
2. **Top-K**: Set `top_k` based on your use case (3-5 for precision, 10+ for recall)
3. **Combine with VoyageAI Embeddings**: For best results, use VoyageAI embeddings with VoyageAI reranker
