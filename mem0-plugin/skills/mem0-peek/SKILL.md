# Mem0 Peek (→ redirected to Tour)

`/mem0:peek` is now built into `/mem0:tour`. Run:

```
/mem0:tour <query>
```

When `/mem0:tour` receives a search query argument, it runs in **peek mode**:
compact one-liner results instead of the full category dump.

This skill exists as a redirect for backward compatibility. Follow the
`/mem0:tour` skill with the user's query argument.
