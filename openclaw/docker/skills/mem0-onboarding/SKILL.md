# Mem0 Memory Onboarding

Welcome! This OpenClaw agent has **persistent memory** powered by Mem0. It remembers things across sessions so you never have to repeat yourself.

## How It Works

- **Auto-Capture**: After each conversation turn, important facts are automatically extracted and stored.
- **Auto-Recall**: Before each turn, relevant memories are retrieved to give context.

## Quick Demo — Try These 3 Steps

### Step 1: Tell me something about yourself
Share a preference, a project you're working on, or a fact about yourself. For example:
> "I prefer TypeScript over JavaScript and I'm building a SaaS app called Acme."

### Step 2: Confirm storage
After you share, I'll confirm that the information was captured. You can also check with:
> "What do you remember about me?"

### Step 3: Verify persistence
Start a **new session** (close this chat and open a new one), then ask:
> "What do you know about me?"

Your memories carry over — that's the magic of persistent memory.

## Power User Commands

If your OpenClaw setup includes the Mem0 CLI commands:

- `openclaw mem0 search "query"` — Search your memories
- `openclaw mem0 stats` — View memory statistics
- `openclaw mem0 list` — List recent memories
