- **数据库核心**：`database.py` 封装了 SQLAlchemy 引擎与会话工厂，支持通过 `DATABASE_URL` 环境变量切换后端（默认 SQLite）。
- **领域模型**：`models.py` 定义了用户、应用、记忆及其关联实体（如分类、访问控制、审计日志），并利用 SQLAlchemy 事件监听器实现记忆插入/更新后的自动 AI 分类。
- **模式定义**：`schemas.py` 使用 Pydantic 定义 API 请求/响应数据结构，处理 UUID 序列化及时间戳转换。
- **迁移管理**：`alembic/` 目录包含完整的迁移历史，支持数据库结构的版本控制与自动演进。
- **外部集成**：`utils/memory.py` 负责 Mem0 客户端的生命周期管理，支持多向量存储后端（Qdrant, Chroma 等）及 LLM 提供商的动态配置与 Docker 环境适配。
