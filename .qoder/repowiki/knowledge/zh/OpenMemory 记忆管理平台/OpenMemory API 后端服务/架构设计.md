- **双协议入口协同**：在 `main.py` 中同时挂载标准 HTTP 路由器（`app/routers`）与 MCP 专用路由器（`app/mcp_server`），前者提供管理端点，后者通过 SSE/Streamable HTTP 暴露给 LLM 客户端。
- **共享基础设施层**：所有模块共用 `app/database.py` 定义的 SQLAlchemy 会话与 `app/models.py` 定义的 ORM 模型，确保关系型数据与向量存储元数据的一致性。
- **统一配置与初始化**：通过 `app/utils/memory.py` 实现 Mem0 客户端的懒加载与环境自适应（如 Docker 网络修正），并在应用启动时执行默认用户与应用实体的自动引导。
- **跨层权限控制**：在 HTTP 路由与 MCP 工具函数中均注入 `app/utils/permissions.py` 逻辑，基于 `AccessControl` 模型实现细粒度的记忆访问隔离。
