- **入口层**：`main.py` 作为 FastAPI 应用启动点，负责数据库表初始化、默认用户/应用创建，并通过 `setup_mcp_server` 注册 MCP 路由。
- **核心逻辑层**：`mcp_server.py` 定义了 MCP 工具（如 `add_memories`, `search_memory`），采用懒加载策略初始化记忆客户端以增强鲁棒性，并通过 `contextvars` 管理多租户上下文（`user_id`, `client_name`）。
- **传输适配层**：实现了 SSE (`/mcp/{client_name}/sse/{user_id}`) 和 Streamable HTTP (`/mcp/{client_name}/http/{user_id}`) 两种传输协议，其中 Streamable HTTP 采用无状态模式并拦截 ASGI 响应以确保兼容性。
- **测试层**：`tests/test_mcp_server.py` 使用 `pytest` 和 `httpx` 的 ASGI 传输对 MCP 协议流程（initialize, tools/list, tools/call）及上下文隔离进行端到端验证。
