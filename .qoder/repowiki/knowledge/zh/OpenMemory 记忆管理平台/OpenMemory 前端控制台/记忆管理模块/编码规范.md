- 采用 URL SearchParams 作为分页和搜索状态的单一事实来源，通过 `router.push` 更新参数以触发数据重新加载。
- 在列表页和详情页均集成全局 `UpdateMemory` 对话框，并通过 `useUI` Hook 统一管理其打开/关闭状态及上下文数据。
- 使用 Redux Toolkit (`useSelector`, `useDispatch`) 管理跨组件的复杂状态，如记忆选中集合、API 加载状态及过滤条件。
- 所有涉及异步数据获取的组件（如 `MemoriesSection`, `MemoryContent`）均在 `useEffect` 中调用自定义 API Hooks，并配套实现 Skeleton 或 Loading 状态以提升用户体验。
