- **组件层**：`components/ui/` 封装了 40+ 个基于 Radix UI 和 Tailwind CSS 的原子化组件（如 `button.tsx`, `dialog.tsx`），通过 `lib/utils.ts` 的 `cn` 函数实现样式合并。
- **状态层**：`store/` 目录采用 Redux Toolkit 架构，通过 `store.ts` 集中配置 `memories`, `apps`, `ui` 等切片，实现全局状态持久化与同步。
- **逻辑层**：`hooks/` 目录下的自定义 Hooks（如 `useMemoriesApi.ts`）负责封装 Axios 请求，处理 API 响应适配并触发 Redux Action，实现视图与数据的解耦。
- **辅助层**：`skeleton/` 提供加载态占位组件，`theme-provider.tsx` 集成 `next-themes` 实现深色模式切换。
