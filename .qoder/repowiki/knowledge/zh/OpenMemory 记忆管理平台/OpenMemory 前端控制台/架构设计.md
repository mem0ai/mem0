- **根布局编排**：`app/layout.tsx` 作为顶层容器，嵌套 `Providers`（Redux Store）、`ThemeProvider`（主题切换）及 `Navbar`（全局导航），确立统一的页面骨架。
- **状态注入**：通过 `app/providers.tsx` 将 Redux Store 注入 React 上下文，确保 `apps_feature` 和 `memories_feature` 等业务模块能共享全局状态（如筛选条件、用户配置）。
- **样式与主题协同**：利用 `next-themes` 与 Tailwind CSS 实现全局限定的深色模式（`bg-zinc-950`），并通过 `globals.css` 统一定制基础样式，保证跨模块视觉一致性。
