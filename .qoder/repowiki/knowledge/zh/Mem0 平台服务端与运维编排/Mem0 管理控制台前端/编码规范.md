- 所有页面组件均采用 Server Components 默认模式，仅在需要交互或访问浏览器 API 时显式声明 "use client" 指令。
- API 请求统一通过 `src/utils/api.ts` 导出的 `api` 实例发起，禁止在业务代码中直接使用原生 fetch 或重复创建 axios 实例。
- 样式类名合并统一使用 `cn` 工具函数（基于 `clsx` 和 `tailwind-merge`），以确保 Tailwind 类名的正确覆盖与组合。
- 表单验证与处理遵循 React Hook Form + Zod 模式，通过 `@hookform/resolvers` 将 Zod schema 集成到表单逻辑中。
