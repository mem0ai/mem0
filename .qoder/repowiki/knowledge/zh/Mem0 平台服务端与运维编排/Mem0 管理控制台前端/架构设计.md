- **框架与路由**：采用 Next.js 15 App Router，通过 `(auth)` 和 `(root)` 平行路由组实现认证页与受保护仪表盘的布局隔离。
- **状态管理**：集成 Redux Toolkit (`@reduxjs/toolkit`) 管理全局 UI 状态（如侧边栏折叠），结合 React Context (`AuthProvider`) 处理用户会话与权限逻辑。
- **数据交互层**：封装 Axios 实例 (`src/utils/api.ts`) 实现统一的请求拦截、JWT Token 自动注入及 401 错误下的静默刷新机制。
- **UI 架构**：基于 shadcn/ui 组件库与 Tailwind CSS 构建响应式界面，利用 `next-themes` 实现深色/浅色模式切换，并通过 `middleware.ts` 实施路由级别的访问控制与初始化检查。
