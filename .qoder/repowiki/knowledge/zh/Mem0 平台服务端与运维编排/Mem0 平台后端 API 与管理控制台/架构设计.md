- 后端采用 FastAPI 框架，通过 `main.py` 作为入口，集成 `routers` 子模块处理 auth、api_keys、entities 和 requests 路由。
- 数据层使用 SQLAlchemy ORM 与 PostgreSQL (pgvector)，通过 `alembic` 进行数据库迁移管理。
- 状态管理采用全局单例模式 (`server_state.py`) 维护 Mem0 核心实例与配置，支持运行时动态更新。
- 前端为 Next.js 15 应用，采用 App Router 架构，通过 `(auth)` 和 `(root)` 路由组实现布局隔离，利用 React Context (`AuthProvider`) 和 Redux Toolkit 管理全局状态。
- 前后端通过 HTTP REST API 通信，前端封装 `axios` 客户端并集成 JWT 自动刷新机制。
