- 后端路由函数统一使用 `Depends(verify_auth)` 或 `Depends(require_admin)` 进行权限校验，并通过 `request.state.auth_type` 记录认证方式。
- 所有可能抛出外部依赖异常的 API 端点均使用 `try...except` 捕获并转换为 `upstream_error()` 或具体的 `HTTPException`。
- 前端 API 请求封装在 `utils/api.ts` 中，并在 `lib/auth.tsx` 中通过 `setAccessToken` 统一管理 Bearer Token 的注入与刷新。
- 敏感配置项（如 API Key、Password）在返回前端前通过 `_redact_config` 函数进行脱敏处理，替换为 '[redacted]'。
