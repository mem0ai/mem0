- 所有命令处理函数（如 `cmd_add`, `cmd_search`）均接收 `Backend` 实例而非直接导入具体实现，以支持潜在的多种后端扩展。
- 在 `app.py` 中通过 `_get_backend_and_config` 统一处理 API Key 的加载、环境变量覆盖及前置 Ping 验证，确保全局认证状态一致。
- 输出逻辑严格区分人类可读模式（text/table）与机器可读模式（json/agent），Agent 模式下自动切换为结构化 JSON 信封格式。
- 配置修改后通过 `plugin_sync.py` 尝试同步 API Key 到生态插件（如 Claude Code），实现跨工具的身份共享。
