- **统一后端抽象**：各子模块均基于 `mem0ai` SDK 或 MCP 协议构建，屏蔽底层存储（Qdrant/PGVector）与模型差异。
- **宿主生命周期编排**：利用宿主平台的钩子系统（如 OpenClaw 的 `before_prompt_build`、Pi Agent 的 `session_start`、Cursor 的 Hooks）实现记忆的自动检索（Auto-Recall）与捕获（Auto-Capture）。
- **技能化记忆管理**：在高级模式下，通过注入系统提示词（System Prompt）触发“Triage”、“Dream”等技能，实现由 Agent 自主控制的记忆整理与去重。
- **隔离与上下文感知**：通过解析会话键（Session Key）与应用 ID，实现跨项目、跨用户及子代理（Sub-agent）的记忆命名空间隔离。
