- **入口与生命周期**：`src/entry.ts` 作为 `pi-coding-agent` 扩展入口，通过 `session_start`、`before_agent_start` 等钩子管理记忆上下文（ScopeContext）和 Dream 触发逻辑。
- **核心功能层**：`src/memory/tools.ts` 封装 `mem0_memory` 工具，提供增删改查能力；`src/commands.ts` 注册 `/mem0-search`、`/mem0-remember` 等 CLI 命令。
- **辅助模块**：`src/capture` 负责对话自动捕获；`src/dream` 实现基于文件锁的记忆整理（去重、修剪）调度；`src/memory/scoping.ts` 处理项目/会话维度的作用域过滤。
- **技能集成**：`skills/` 目录包含多个 `SKILL.md` 文件，为 Agent 提供记忆操作的自然语言引导。
