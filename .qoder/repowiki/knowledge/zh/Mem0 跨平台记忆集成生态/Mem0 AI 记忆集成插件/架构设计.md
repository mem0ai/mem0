- **多宿主适配层**：通过 `plugin.json`、`.cursor-mcp.json`、`.codex-mcp.json` 及 `.opencode-plugin/` 目录下的配置，分别对接不同 IDE/CLI 的插件系统与 MCP（Model Context Protocol）标准。
- **生命周期钩子系统**：利用 `hooks.json` 定义事件驱动架构，在 `SessionStart`、`UserPromptSubmit`、`Stop` 等关键节点触发 Bash/Python 脚本（如 `on_session_start.sh`），实现记忆的自动注入与持久化。
- **OpenCode 深度集成模块**：在 `.opencode-plugin/` 中采用 TypeScript 编写核心逻辑 (`opencode-mem0.ts`)，直接调用 `mem0ai` SDK 并监听 `chat.message`、`tool.execute` 等底层事件，实现细粒度的上下文干预。
- **技能与工具生态**：通过 `skills/` 目录提供标准化的 `SKILL.md` 文档，指导 AI 使用 `/mem0:` 系列命令；同时封装 Python 辅助脚本库（位于 `scripts/`）处理身份解析、依赖安装及元数据强制校验。
